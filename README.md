# **腕上OSU身份卡**

## 什么是腕上osu身份卡
  腕上osu身份卡是一款可以在Xiaomi Vela手表和手环上展示您的osu账户信息和成绩的App，目前仍处于开发中。

## 目前可以在哪些设备上运行
  所有具备网络能力的小米 Vela 穿戴设备（包括蓝牙网络连接和蜂窝网络连接），具体如下
- 小米手环 8Pro
- 小米手环 9/9Pro
- 小米手环 10
- 红米手表 5/5esim
- 红米手表 6
- 小米手表 s3/s4/s5 系列

请注意，目前 UI 界面上仅对方屏手表/手环进行了适配，圆形屏幕及药丸屏请等待后续开发。

## 如何使用

1. 前往osu.ppy.sh，登录您的osu账号
2. 点击头像登录并进入个人主页，并查看您的地址栏，地址栏中users后的一串数字是您的UID，记录下您的UID
3. 点击头像，在弹出菜单中点击设置，进入设置页面，这一步通常会要求验证您的邮箱
4. 选择账户设置，拉到页面最下方，找到旧版本 API
5. 点击新的旧版本 API 密钥，生成属于你的 API Key，应用名称可自定义，应用 URL 填写本项目地址，并点击创建密钥
6. 此时会有显示密钥按钮，点击显示密钥，将 API 密钥字段记录下
7. 打开本应用，在设置中填入您的 UID 和 API 密钥，记得进行测试与保存
8. 退出应用，再次进入，就能看到你的用户信息了

需要注意的是，API 密钥有调用次数限制，请注意关注 osu 网站页面上的信息，避免频繁刷新导致密钥达到限额。

请保管好自己的 API 密钥
